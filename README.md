# TEDxMET Nashik — Speaker Nominations 2026

A web application for nominating exceptional speakers for TEDxMET Nashik 2026. We're looking for thinkers, founders, artists, researchers, and storytellers with ideas worth spreading.

## Features

- **Speaker Nomination Form**: Submit nominations for speakers
- **Responsive Design**: Works seamlessly on all devices
- **SEO Optimized**: Built with search engine optimization in mind
- **Interactive UI**: Smooth animations and user-friendly interface

## Project Structure

```
.
├── index.html       # Main HTML file
├── style.css        # Styling
├── script.js        # JavaScript functionality
├── assets/          # Images, icons, and other media
└── README.md        # This file
```

## How to Use

1. Clone this repository
2. Open `index.html` in your web browser
3. Fill out the nomination form and submit

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling and responsive layout
- **JavaScript**: Client-side functionality
